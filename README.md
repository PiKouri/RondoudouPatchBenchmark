# Running RondoudouPatchBenchmark
```console
git clone https://github.com/PiKouri/RondoudouPatchBenchmark
cd RondoudouPatchBenchmark/src
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
CipherBenchmarks/CipherBenchmarks
```
